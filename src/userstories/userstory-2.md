# Example Dataset Holder: A User Story

Example user story for dataset holders. As a dataset holder, how to best manage access to important patient-level data while protecting the integrity and privacy of the individuals included in the dataset?

- How are requests handled by dataset holders today?
- What does the LGF provide to assist the dataset holder to manage linkage requests?
- How does the common governance process smooth the way for linkage approvals?
- How does it reduce burden on the dataset holder?

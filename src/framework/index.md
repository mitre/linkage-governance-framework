# What problem does this framework address?

Linkage contends with each dataset’s unique decision making processes and policies that direct permitted use and protect privacy. The process and policy challenges when linking multiple datasets can include:

- Investigators must submit separate requests to multiple organizations which is time-consuming, redundant, and requires intense up-front effort 
- Organizations receive requests for data access & linkage that are incomplete which results in additional time-consuming iterations with Investigators to clarify technical details
- Review processes and approvals are sequenced differently across organization which creates rework and inefficiency for an Investigator 
- Existing organizational review and decision-making processes do not facilitate collaboration between multiple data contributing organizations
- Investigators may gain approvals to access data without a plan for how to implement the linkage   
- Investigators may learn that conflicting governance rules make linkage impossible with no way to reconcile these conflicts
- No process or tools exist to review governance rules that apply across datasets and define a set of rules to govern the linked dataset 

The Linkage Governance Framework is a new model for governance that confronts many of the challenges that delay the creation of linked data for research.

## What could this framework achieve?

This framework’s mission is to enable complex linkages with greater flexibility[^1] and scalability[^2] than is currently available. Enabling complex linkages translates to more research studies linking data are initiated, more linkages of many datasets are completed faster, and complex linkages can be completed for less costs. 

[^1]: Flexibility means that the framework is built to facilitate the harmonization of many data types and data collected for purposes other than research.
[^2]: Scalability means that the framework is built to create complex linkages for research, public health, and population health applications drawing data from federal, academic, and other institutions.
